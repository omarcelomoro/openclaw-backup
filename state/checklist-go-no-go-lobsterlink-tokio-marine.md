# Checklist GO / NO-GO — LobsterLink para Tokio Marine

Data: 2026-04-19

## GO se
- o bloqueio principal for autenticação humana
- o LobsterLink permitir completar login/2FA/CAPTCHA
- a sessão autenticada permanecer ativa no browser do agente
- o agente conseguir navegar depois para a área de interesse
- a manutenção adicional for aceitável

## NO-GO se
- o browser normal já resolver o fluxo
- o problema principal não for autenticação humana
- a extensão trouxer instabilidade relevante
- o viewer falhar com frequência
- a sessão autenticada não se sustentar
- o ganho operacional for pequeno demais para a complexidade adicionada

## INCONCLUSIVO se
- o fluxo testado não representar o uso real
- o bloqueio variar demais entre tentativas
- a navegação pós-login ainda não estiver bem definida
- faltar clareza sobre quais dados realmente precisam ser extraídos

## Recomendação atual
Estado atual: **candidata forte para piloto restrito, ainda não aprovada como padrão da Leevre**.
