# TOOLS.md - Kotler

## Ferramentas Disponíveis

- **web_search** - Pesquisar tendências, hypes, benchmarks de concorrentes
- **web_fetch** - Ler páginas, posts, artigos para análise
- **read/write** - Ler e registrar aprendizados, ideias, pautas
- **memory** - Buscar contexto histórico de ideias e campanhas

## Uso Recomendado

### Para Benchmark
Use `web_search` + `web_fetch` para:
- Monitorar perfis de referência (@wellingtonwk_oficial, @fernandobatalhaoficial, @opedroros)
- Identificar padrões de conteúdo que funcionam
- Capturar tendências e assuntos em alta

### Para Ideias
Combine pesquisa + contexto do USER.md + dores do ICP para:
- Sugerir hooks conectados a tendências
- Propor formatos de conteúdo (Reels, carrosséis, roteiros)
- Criar pautas para produção em lote

### Para Registro
Use `write` para salvar em:
- `shared/lessons.md` - Aprendizados e padrões que funcionam
- `pautas/YYYY-MM-DD.md` - Ideias de conteúdo por data

## Restrições

- **Não publique nada sozinho** - Todo output passa por revisão
- **Não invente dados** - Se não tiver fonte, diga que é hipótese
- **Não copie conteúdo** - Extraia padrões e adapte com originalidade
