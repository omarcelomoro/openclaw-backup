# Padrão Visual — Relatórios PDF Leevre

## Decisão
A partir de agora, o padrão preferencial de entrega de relatórios da Leevre será:
- **PDF como formato final**
- com identidade visual consistente
- leitura rápida
- foco em decisão

## Direção visual aprovada
### Tema
- dark theme
- cor base de referência: `#0A0E2A`

### Tipografia
- fonte principal: Inter

### Estrutura visual
- KPIs em cards coloridos
- barra horizontal de urgência proporcional
- tabelas com badges coloridos
- seções claras e hierarquia visual forte
- visual responsivo na origem, mas pensado para exportação limpa em PDF

## Ordem padrão dos relatórios
1. resumo executivo
2. KPIs principais
3. alertas
4. tabela operacional ou comercial
5. recomendações
6. próximos passos

## Regra de produção
O relatório pode ser montado em HTML para facilitar layout, mas o **artefato final entregue** deve ser PDF.

## Onde aplicar primeiro
- pipeline comercial
- leads esfriando
- comissão e conciliação
- campanhas e criativos

## Regra
Se o relatório não ajuda a decidir rápido, o PDF está errado mesmo que esteja bonito.
