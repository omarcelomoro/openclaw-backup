# Geração de Relatórios PDF — Leevre

## Regra aprovada
Sempre que a Livri gerar relatório operacional, comercial ou de marketing para a Leevre, o formato final preferencial deve ser **PDF**.

## Padrão visual
- dark theme
- Inter
- cards coloridos
- barra de urgência
- badges de prioridade
- hierarquia visual clara

## Fluxo recomendado
1. montar layout em HTML standalone
2. revisar clareza, ordem e destaque visual
3. exportar para PDF
4. entregar o PDF como artefato final

## Ordem padrão do conteúdo
1. resumo executivo
2. KPIs principais
3. alertas
4. tabela principal
5. recomendações
6. próximos passos

## Uso inicial recomendado
- relatório de pipeline
- relatório de leads esfriando
- relatório de comissão e conciliação
- relatório de campanhas
