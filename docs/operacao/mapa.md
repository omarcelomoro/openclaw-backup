# Mapa — Operação Leevre

## Objetivo desta área
Centralizar a documentação operacional da Leevre em uma estrutura que seja fácil de navegar, consultar e evoluir.

## O que este mapa resolve
- mostra as áreas operacionais já documentadas
- indica por onde começar em cada frente
- ajuda a transformar conhecimento solto em sistema acessível
- reduz dependência de memória de chat ou servidor

## Padrão da documentação operacional
Cada área deve ter:
- `mapa.md`
- `00-comece-aqui.md`
- `README.md`

## Como usar esta estrutura
- se quiser visão geral: comece por este `mapa.md`
- se quiser entrar numa área: abra o `mapa.md` da área
- se quiser executar rápido: abra o `00-comece-aqui.md`
- se quiser índice formal: abra o `README.md`
