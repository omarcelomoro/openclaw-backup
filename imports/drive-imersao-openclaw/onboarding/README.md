# Onboarding — Configure o Cérebro da Sua Empresa

Esta pasta contém o guia de setup para transformar este repositório — que hoje tem dados de uma empresa fictícia — no cérebro da sua empresa.

## Como começar

Clone o repositório e cole este comando no seu agente:

```
Leia onboarding/SETUP.md e configure o cérebro da minha empresa
```

O agente vai guiar você por todo o processo. Tempo estimado: 60–90 minutos.

## O que o setup faz

1. **Limpa os dados da empresa de demonstração** — mantendo toda a estrutura de pastas
2. **Preenche o contexto da sua empresa** — quem você é, equipe, métricas
3. **Configura a primeira área** — vendas, marketing, atendimento ou operações
4. **Cria a primeira skill** — automação da sua tarefa mais repetitiva
5. **Agenda o primeiro cron** — skill rodando sozinha no horário certo
6. **Publica no GitHub** — seu cérebro versionado e pronto para usar
