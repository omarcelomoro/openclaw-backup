# Vendas — Mapa da Área

## Estrutura

```
vendas/
├── MAPA.md
├── contexto/
│   ├── geral.md           ← Objetivo, KPIs, funil, ferramentas
│   ├── people.md          ← Quem trabalha na área
│   ├── decisions.md       ← Decisões da área
│   └── lessons.md         ← Lições aprendidas
├── rotinas/
│   ├── relatorio-vendas-diario.md
│   └── follow-up-leads-diario.md
├── skills/
│   └── _index.md
├── projetos/
├── bot/
│   └── conhecimento.md
```

## O que tem em cada lugar

| Caminho | O que o agente encontra |
|---------|------------------------|
| `contexto/geral.md` | Objetivo da área, KPIs, funil, regras comerciais |
| `contexto/people.md` | Responsáveis e estrutura de decisão |
| `contexto/decisions.md` | Decisões tomadas na área |
| `contexto/lessons.md` | Lições aprendidas |
| `rotinas/` | Crons configurados (relatório diário 8h, follow-up 9h) |
| `skills/` | Skills da área (ver `_index.md`) |
| `projetos/` | Projetos ativos e concluídos |
| `bot/` | Base de conhecimento do bot de vendas |
