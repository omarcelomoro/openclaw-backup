- **MEMORY.md = índice local.** Aponta para onde a informação realmente vive.
- **Notas diárias = rascunho.** Consolidar no repositório e deletar.
- **Decisão de campanha?** → `cerebro/areas/marketing/sub-areas/trafego-pago/`
- **Dado de performance?** → `cerebro/areas/marketing/sub-areas/trafego-pago/`
