# analise-concorrentes-conteudo

## Objetivo
Modelar inteligência de conteúdo a partir de referências fortes do mercado.

## Verificar
- ICP implícito
- dores ativadas
- hooks usados
- promessa central
- tom de autoridade
- padrão de conversão

## Saída esperada
- aprendizados acionáveis
- ângulos adaptáveis para a Leevre
- riscos de conteúdo genérico ou copiado
