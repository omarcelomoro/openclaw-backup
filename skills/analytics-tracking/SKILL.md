# Analytics Tracking

Set up tracking that provides actionable insights for marketing and product decisions.

## Core principles
- Track for decisions, not vanity
- Start with the business questions
- Name events consistently
- Protect data quality

## Tracking plan framework
Define:
- business context
- current state
- technical context
- event names
- properties
- triggers
- notes

## Naming convention
Prefer object-action names like:
- `signup_completed`
- `form_submitted`
- `button_clicked`
- `checkout_payment_completed`
