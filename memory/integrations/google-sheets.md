# Google Sheets — mapa operacional

**Papel:** base prática para controle, importação e conferência rápida.
**Status atual:** relevante na operação; ainda faltam integrações mais robustas em alguns fluxos.

## Para que serve hoje
- base de clientes
- renovações
- importações/exportações
- consolidação provisória de dados

## O que já temos
- planilhas com base histórica de clientes
- uso operacional real no dia a dia

## O que falta
- padronizar planilhas que viram fonte oficial
- separar planilha de controle provisório vs base confiável
- melhorar ponte com HubSpot e rotinas futuras

## Riscos
- duplicidade de verdade entre planilha, CRM e memória
- campos sem padrão
- manutenção manual sem critério

## Próxima revisão sugerida
Quando a exportação/base de seguros for retomada.
